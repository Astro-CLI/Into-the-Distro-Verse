Shell Configuration Guides

Complete guides for configuring Bash, Zsh, and Fish shells with modern CLI tools and features.

Files

  • bash.md - Bash configuration guide with readline, history, and CLI tool integration
  • zsh.md - Zsh configuration guide with Oh My Zsh, plugins, and keybindings
  • fish.md - Fish shell guide with native features, functions, and POSIX notes

Quick Links

All three shells are configured with: - Emacs keybindings (Ctrl+A, Ctrl+E, Ctrl+K, etc.) - Zoxide for directory jumping - Thefuck for command correction - FZF for fuzzy finding and history search - Starship for unified prompt

For backup and restore instructions, see configs/shells/README.md
